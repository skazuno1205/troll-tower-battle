# Troll Tower Battle

`Troll Tower Battle` は、トロールを積み上げてハイスコアを狙う 1 人用ブラウザゲームです。  
見た目が違うトロールたちをバランスよく落とし、崩さずにどこまで高く積めるかを競います。

## ゲームの要素
- **スタック**: トロールを中央付近に落とすほど安定し、連続成功でコンボが発生します。
- **コンボ / FEVER**: きれいに積み続けると FEVER ゲージが溜まり、プレイを有利に進められます。
- **アイテム**: プレイ中に入手できるアイテムを使ってピンチを切り抜けられます。
- **ベスト記録**: 最大積み上げ数はブラウザの `localStorage` に保存されます。

## 操作方法
- **マウス / タッチ移動**: 落下位置の調整
- **回転ボタン**: トロールの角度調整
- **DROP**: トロールを落とす
- **ITEM**: 所持アイテムを使用
- **RESTART**: リトライ

## 起動方法
1. このリポジトリを開く
2. `index.html` をブラウザで開く
3. `START` を押してゲーム開始

## ファイル構成
- `index.html`: UI レイアウトとゲーム画面の土台
- `style.css`: 画面デザイン
- `game.js`: ゲームロジック・物理挙動・描画処理
- `images/`: トロール画像アセット

## メモ
- ビルド不要の静的 Web ゲームです。
- 変更を確認する場合はブラウザ再読み込みで反映されます。
