# troll-tower-battle

`troll-tower-battle` は、ブラウザで遊べるタワーバトルゲームのプロトタイプです。

## ファイル構成
- `index.html`: エントリーポイントとなるHTML。
- `troll_tower_battle_v32.html`: ゲーム本体（ロジックとUIを含む）。

## 遊び方
1. `index.html` をブラウザで開きます。
2. 画面の案内に従ってユニットを配置し、相手の拠点を攻略します。

## 開発メモ
- 追加のビルド手順は不要で、静的ファイルとしてそのまま動作します。
- 修正後はブラウザを再読み込みして反映を確認してください。
